# Stay Near Bedford Universal

**The UK's complete independent guide to Universal Studios Bedford — opening 2031.**

> Live site: [staynearbedforduniversal.com](https://www.staynearbedforduniversal.com)

## About

A single-page informational website covering everything visitors need to know before Universal Studios Bedford opens:

- 🏨 **Accommodation** — Hotels, Airbnb, glamping and holiday lets near the park
- 🚆 **Travel guides** — Trains from London, Luton Airport transfers, taxis and coaches
- 🍽️ **Dining** — Best restaurants, pubs and cafes near Kempston Hardwick
- 📰 **News & updates** — Construction progress and official announcements
- 💼 **Jobs** — Careers information for the 28,000 roles being created
- ❓ **FAQ** — Answers to the most common questions about the park

## Development

This is a standalone HTML file — no build step required. Simply open `index.html` in a browser or deploy to any static hosting service (GitHub Pages, Netlify, Vercel, etc.).

## Deployment (GitHub Pages)

1. Push to GitHub
2. Go to **Settings → Pages**
3. Set source to the `main` branch, root directory
4. Your site will be live at `https://<username>.github.io/<repo-name>/`
